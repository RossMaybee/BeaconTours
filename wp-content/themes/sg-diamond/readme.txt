## About SG Diamond

Description: SG Diamond is a responsive theme with many custom options. It has number of widgets (slider, social icons, portfolio, sidebar widget for creating custom sidebars, recent/popular posts). Supports portfolio from Jetpack and WooCommerce, translation ready.

## How to install

* Unzip and then copy the layout-builder directory to /wp-content/themes/ in your WordPress installation. 
* Activate from admin menu. 

## How to use

* Theme options placed at "Appearance >> Customize"
* Widgets can be found at "Appearance >> Widgets"
* Documentation: http://wpblogs.ru/themes/how-to-video-sg-window-theme/
* SG Diamond works with WordPress 4.2 and higher. Be sure you use latest version of WordPress.
